# LoakPedia

LoakPedia is a **Laravel-based web application** designed as a marketplace platform for second-hand (pre-owned) items.  
This project focuses on clean backend architecture, dynamic web rendering, and proper Laravel project structuring.

The application was developed as a **portfolio project** to demonstrate practical skills in **Laravel, PHP, and web application development**.
